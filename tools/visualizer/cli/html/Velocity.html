Velocity

A simple measure of how much we deliver each sprint. The graph shows a moving average of how many user stories are completed each sprint.

Reading the graph

If we are improving there will be an upwards trend. The two lines represent a shorter and a longer persepctive. The trend of the "long" is a stronger signal however if the short has recently changed direction, to the opposite of the long trand, this might be a sign that the long trend will change in the near future

Method

We can improve this number by either being more efficient at want we do or by creating smaller user stories. Both are something that we want to do. Since there are multiple ways to impact the graph we won't know to what extend they both influence the change but only that an upwards trend is positive.
